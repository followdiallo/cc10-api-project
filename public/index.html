POST

/api/queens/
Adds a new queen with the information specified in the request body.

PATCH

/api/queens/:name
Updates the information of the selected queen. Note that names are case-sensitive, and you must indicate the queen's full name.

DELETE

/api/queens/:name
Deletes the selected queen. Note that names are case-sensitive, and you must indicate the queen's full name.


âœ¨ABOUT THE DATA âœ¨

  • A lot of the data comes from the RuPaul-Predict-A-Looza Spreadsheet.
